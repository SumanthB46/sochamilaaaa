Display the jersey as flat-lay or on a neutral mannequin

Use a plain neutral background (white, gray, or studio background)

Front View (Image 1 must show):

Primary and secondary colors clearly

Collar type (round, V-neck, polo, etc.)

Sleeve design and patterns

Team logo placement

Sponsor text or logo (if provided)

Back View (Image 2 must show):

Player name clearly printed

Jersey number in bold, readable font

Back panel or side design elements

Same fabric and color alignment as front

Quality Standards:

Photorealistic lighting and shadows

High sharpness and clarity

No watermarks, no extra text, no background objects
